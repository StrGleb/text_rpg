# Текстовая ASCII-RPG «Спасение Короля» на C++

Учебный проект, разработанный в рамках летней производственной практики. Игра представляет собой пошаговую приключенческую CLI-аркаду в стиле файтинга (Mortal Kombat) с фиксированным кадром отрисовки.

## 🚀 Ключевые особенности проекта
* **Объектно-ориентированная архитектура (ООП):** Активное применение наследования, инкапсуляции, полиморфизма и композиции.
* **Data-Driven Design:** Характеристики противников и параметры игрового лута полностью вынесены во внешние конфигурационные файлы конфигурации (`.txt`), что позволяет менять баланс игры без перекомпиляции кода.
* **Уникальные механики классов:** Реализовано три игровых класса (Рыцарь, Маг, Лучник), каждый из которых обладает уникальными боевыми характеристиками и особым тактическим навыком (поглощение урона, контратака, гарантированный крит).
* **Визуализация файтинга:** Динамические полоски здоровья (HP Bars), отображение уровня персонажа и анимация ударов с помощью проработанного ASCII-арта.
* **Устойчивость к ошибкам (Ввод):** Интегрирована валидация потока `std::cin` для защиты от некорректного ввода (букв вместо цифр).
* **Система лидеров:** Автоматический подсчет очков и сохранение ТОП-5 лучших результатов в файл рекордов.

## 📂 Структура проекта
```text
text_rpg/
├── config/              # Текстовые конфигурации баланса и рекордов
│   ├── items.txt        # База данных лута
│   ├── monsters.txt     # База данных противников
│   └── highscores.txt   # Файл таблицы лидеров
├── core/                # Ядро игры (Логика персонажей)
│   ├── Character.h/cpp  # Базовый класс живых существ, криты
│   ├── Player.h/cpp     # Класс героя, опыт, механики навыков
│   └── Monster.h/cpp    # Класс монстров и финального Босса
├── items/               # Модуль инвентаря
│   ├── Item.h/cpp       # Описание предметов (зелья, оружие)
│   └── Inventory.h/cpp  # Управление вектором слотов сумки
├── graphics/            # Модуль визуализации
│   └── Renderer.h/cpp   # Очистка экрана, тайминги, ASCII-арт сцен
├── scores/              # Модуль таблицы рекордов
│   └── Highscores.h/cpp # Сортировка и потоковое сохранение очков
├── main.cpp             # Точка входа и главный игровой цикл
└── .gitignore           # Исключение артефактов сборки из Git
```

### 🛠️ Сборка и запуск

### 🐧 Linux (Ubuntu) & 🍏 macOS (Terminal)
Находясь в корневой директории проекта, выполните универсальную команду сборки всех модулей через маску `*.cpp`:
```bash
g++ main.cpp core/*.cpp items/*.cpp graphics/*.cpp scores/*.cpp -o game && ./game
```
### 🪟 Windows (Command Prompt / PowerShell)
В Windows стандартная маска `*.cpp` через встроенную консоль не поддерживается. Для сборки необходимо явно перечислить все файлы проекта:

**Через командную строку (cmd):**
```cmd
g++ main.cpp core/Character.cpp core/Player.cpp core/Monster.cpp items/Item.cpp items/Inventory.cpp graphics/Renderer.cpp scores/Highscores.cpp -o game.exe && game.exe
```

**Через PowerShell:**
```powershell
g++ main.cpp (Get-ChildItem core/*.cpp, items/*.cpp, graphics/*.cpp, scores/*.cpp | Select-Object -ExpandProperty FullName) -o game.exe; .\game.exe
```
